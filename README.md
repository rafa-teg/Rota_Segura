# Rota_Segura
Rota Segura - Monitoramento e Geolocalização Nativa 
O Rota Segura é um módulo mobile desenvolvido para oferecer rastreamento em tempo real e visualização de pontos de interesse (como ecopontos e rotas de coleta) integrando mapas e serviços de localização.

📱 Funcionalidades Principais
Integração com Google Maps: Visualização dinâmica de mapas e marcação de coordenadas específicas.

Geolocalização em Tempo Real: Gerenciamento de permissões e captura de localização do usuário para traçado de rotas.

Interface Moderna: Desenvolvido com Jetpack Compose, utilizando componentes reutilizáveis para garantir uma navegação fluida.

Customização de UI: Implementação de temas personalizados e tipografia estruturada.

🛠️ Tecnologias Utilizadas
Linguagem: Kotlin (Nativo).

Toolkit de UI: Jetpack Compose.

APIs: Google Maps SDK for Android.

Arquitetura: Componentização de interface para maior manutenibilidade do código.
